## 🔍 Overview

**Berrymind**는  
온실 환경에서 발생하는 **센서 데이터, 작물 생육 상태, 환경 제어 이력**을 통합하여  
**CO₂ 배출량 및 온습도 예측과 지능형 환경 제어 의사결정**을 수행하는 스마트농업 시스템입니다.

본 시스템은  
- 센서 기반 **환경 인지(Sensing)**  
- AI 기반 **환경·CO₂ 예측(Prediction)**  
- **SAM3 기반 작물 생육 단계 인식(Vision)**  
- 규칙 기반 **제어 의사결정(Rule Engine)**  

을 하나의 파이프라인으로 결합하여,  
**실제 스마트팜 운영 환경을 반영한 구조적·확장 가능한 설계**를 목표로 합니다.
